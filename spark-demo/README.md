# SPARK v1 — Demo (showcase + interactive sandbox)

A Next.js showcase and interactive sandbox for the SPARK autonomous rebuild pipeline.
It's a canned replay of a real rebuild — no live Microsoft 365, no credentials,
static fixtures only. The sample org "Crestfield Foods" is fictional.

Deployed live at **https://spark-demo.codeandcraft.ai**.

## Run locally

```bash
npm install
npm run dev   # http://localhost:3000
```

Routes: `/` (showcase / landing) and `/sandbox` (interactive replay).

The autonomous pipeline engine is proprietary and not included in this repo.
